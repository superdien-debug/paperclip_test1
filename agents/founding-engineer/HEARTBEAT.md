# HEARTBEAT.md -- Founding Engineer Heartbeat Checklist

Run this on every heartbeat.

## 1. Identity and Context
- Confirm your ID and role.
- Check wake context and current assignments.

## 2. Engineering Tasks
- Review `todo` and `in_progress` engineering tasks assigned to you.
- Prioritize based on the company roadmap.
- Checkout before working.

## 3. Implementation
- Apply surgical changes to the codebase.
- Write tests for every new feature or bug fix.
- Ensure all CI/CD checks pass.

## 4. Documentation and Updates
- Update issue status and comment on progress.
- Document architectural changes and technical specs.

## 5. Exit
- Comment on any `in_progress` work before exiting.
